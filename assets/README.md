# Assets — ρίξε εδώ τα πραγματικά σου αρχεία

Το `business.html` (και αργότερα το `index.html`) ψάχνει αυτόματα αυτά τα αρχεία.
Μόλις τα προσθέσεις με **ακριβώς αυτά τα ονόματα**, εμφανίζονται στη θέση των
placeholder — χωρίς καμία αλλαγή κώδικα.

| Αρχείο | Τι είναι |
|---|---|
| `logo.png` (ή `.svg`) | Το λογότυπο Bonusaki (η κίτρινη πινακίδα με τα αστέρια) |
| `screen-customer.png` | Το mockup της οθόνης πελάτη (Σκάναρε & κέρδισε) |
| `screen-merchant.png` | Το mockup του dashboard καταστήματος |
| `screen-cashier.png`  | Το mockup της οθόνης ταμείου (ΕΓΚΥΡΟ) |
| `icon-burger.png` `icon-fries.png` `icon-pizza.png` `icon-dessert.png` `icon-soda.png` `icon-coffee.png` `icon-qr.png` `icon-wallet.png` | Τα εικονίδια φαγητού (κομμένα από το asset sheet) |

> Αν έχεις το **ένα ενιαίο asset sheet** (όλα τα εικονίδια μαζί), ρίξ' το ως
> `sheet.png` και πες μου — το κόβω εγώ στα παραπάνω επιμέρους αρχεία.

Υποστηρίζονται `.png`, `.jpg`, `.webp`, `.svg`.
